# EVE Kit

[eve-kit.com](https://eve-kit.com) — EVE Online 工业工具，纯前端 SPA，支持国服 (Serenity) 和世界服 (Tranquility)。

## 功能概览

### 首页 — 服务器状态与战场热点

- 双服务器实时在线人数、在线/离线状态
- 过去 1 小时全服舰船击杀数
- PvP 热门区域 Top 10（舰船 + 太空舱击杀，按区域排名）
- PvE 热门区域 Top 10（NPC 击杀，按区域排名）
- 主权战役实时展示（星系、防守联盟、攻防进度条）
- 区域名中英文跟随语言设置

### 工业制造计算器

- **蓝图搜索**：支持中英文搜索，自动过滤特别版/AT 奖励舰船
- **制造队列**：添加多个产品，设置 ME（材料效率）和流程数
- **递归 BOM 展开**：自动展开最多 5 级材料链（T2 制造→T2 组件→反应中间体→月矿原料）
- **Tab 分级展示**：各级材料通过 tab 切换，每个 tab 显示卖单/收单价格和生产时间
- **吉他实时订单价**：每级显示卖单/收单总价（缓存1小时），最终产品和原材料汇总均显示价格
- **生产时间估算**：显示理论最佳时间（满技能+最佳建筑）到无加成时间范围，最终产品显示总时间汇总
- **已有材料管理**：每级和原材料汇总均支持"已有"弹窗，左侧粘贴库存，右侧实时显示需求/已有/还需
- **一键复制还需材料**：`材料名\t数量` 格式，可直接在游戏中导入
- **智能复制**：选中表格材料后 Ctrl+C，自动格式化为 `材料名\t数量`（无逗号）
- **右键跳过材料**：右键材料置灰+删除线，不再分解，右键恢复
- **子组件 ME**：全局设置，默认 0，一键应用到所有子组件（第一级支持单独设置）
- **分享与导入**：导出产品名+流程数+ME，导入分享文本或直接输入 `产品名\t数量`
- 自动跳过燃料块和 PI 材料展开

### 市场价格查询

- **价格查询**：粘贴材料清单，查询吉他收单/卖单价格，显示单价和小计
- **化矿计算**：粘贴矿石/冰矿/月矿/废铁物品，配置矿石效率和废铁效率，计算化矿产物及吉他卖单总价
- 正确处理新旧压缩矿（portionSize 修正）

### 行星开发计算器 (PI)

- PI 产品制造链计算
- 输入/输出材料展示
- 中英文物品名支持

### LP 商店计算器

- NPC 军团 LP 商店浏览
- LP 兑换物品查询
- ISK/LP 比率计算

### 装配浏览器

- 粘贴游戏内装配格式
- 解析并展示舰船装配方案

### D-Scan / Local 解析

- 粘贴 D-Scan 或 Local 数据
- 解析舰船类型和数量统计

### 旗舰跳跃规划

- 起始/目的地星系搜索（自动补全，中英文）
- 自定义跳跃范围（光年），支持预设：无畏 3.5 / 航母 5.0 / 超旗 6.5 / 跳货 7.0 / 泰坦 10.0 / 座头鲸 15.0
- BFS 最短路径算法，支持旗舰跳跃 + 星门跳跃混合路线
- 高安目的地双路线选项
- 星系规避功能（点击"规避"按钮排除星系）
- 路线表格：跳跃序号、类型、星系、安全等级（颜色标记）、距离、区域

### 公开合同查询

- 按区域搜索（自动补全），按类型筛选（物品交换/拍卖/快递）
- 合同列表：类型、标题、价格、体积、发布/到期日期
- 展开详情：合同 ID、发布者、可见性、物品清单（缓存，秒开）
- 吉他收单对比，折扣率计算（低于收单标绿，溢价标红）
- 复制游戏内链接
- 分页浏览

### 虫洞系统查询

- 虫洞名称搜索、等级筛选（C1-C6）、效应筛选
- 结果表格：星系名、等级、效应、静态洞口、安全等级、区域
- 系统详情：静态洞口类型、目标等级、存续时间、总质量、单跳质量

### 通用功能

- **双语支持**：中文/英文完整翻译，首次访问自动检测浏览器语言
- **双服务器**：国服 (Serenity) / 世界服 (Tranquility) 一键切换，路由自动适配
- **捐赠弹窗**：中文显示微信赞赏码，英文显示 Ko-fi
- **反馈入口**：直接链接到 GitHub Issues
- **致谢弹窗**：数据来源和法律声明
- **EVE 风格 UI**：深黑背景、金色强调色、物品图标、安全等级颜色编码、全局自定义滚动条
- **订单价格缓存**：ESI 订单价格缓存 1 小时，减少重复请求

## 数据来源

| 来源 | 用途 |
|------|------|
| [Fuzzwork SDE](https://www.fuzzwork.co.uk/dump/latest/) | 物品、蓝图、星系、星门等静态数据（CSV → JSON） |
| [CCP ESI](https://developers.eveonline.com/) | 市场价格、订单、合同、服务器状态、击杀统计、主权数据、LP 商店 |
| [Serenity ESI](https://ali-esi.evepc.163.com) | 国服中文物品名和地图名 |
| [eve-bookmarks](https://github.com/OkYk/eve-bookmarks) | 虫洞系统效应和静态洞口数据 |

## 技术栈

| 层 | 技术 |
|----|------|
| 框架 | Vue 3 (Composition API + `<script setup>`) |
| 路由 | Vue Router |
| 状态 | Pinia |
| 构建 | Vite |
| 部署 | Cloudflare Pages (GitHub Actions 自动部署) |
| 数据 | 浏览器直连 ESI + 预构建 JSON |

纯前端 SPA，无后端依赖，可部署到任何静态托管服务。

## 本地开发

```bash
npm install
npm run dev          # 启动开发服务器 (端口 5174)
npm run build        # 构建生产版本
```

## 数据更新

```bash
# 下载最新 SDE 并生成 JSON（含国服中文名）
node scripts/convert-sde.mjs --download --fetch-zh-names

# 生成 LP 商店数据（可选，需要较长时间）
node scripts/convert-sde.mjs --download --fetch-zh-names --fetch-lp
```

## 每周维护

```bash
git pull
node scripts/convert-sde.mjs --download --fetch-zh-names
git add public/data/
git diff --cached --quiet || git commit -m "Weekly SDE data update" && git push
```

Push 到 main 后 GitHub Actions 自动构建部署到 Cloudflare Pages。
